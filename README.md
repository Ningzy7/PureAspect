# PureAspect · 原屏魔术手

> 自动修复游戏从小窗切回全屏时的画面比例扭曲。

## 装好就能用

模块内置了 **80+ 热门游戏包名**（王者、和平精英、原神、金铲铲等），装完重启即生效，**不用配任何东西**。

## 想添加其它游戏？

编辑 `/data/adb/modules/pure_aspect/game_list.conf`，去掉对应游戏前的 `#` 号即可。也可以自己加一行新的包名。

包名查法：酷安/豌豆荚的应用详情页 URL 里就有，或装一个「App 包名查看器」。

## 验证是否在工作

```bash
logcat -s GAFix
```

看到 `从小窗→全屏，触发渲染重建` 就对了。

## 卸载

Magisk App → 模块 → PureAspect → 移除 → 重启。

## 停止监控

```bash
su -c "killall game_aspect_fix_daemon.sh"
```

## 常见问题

**装了卡顿吗？** → 不会，游戏没开时 ≈ 零耗电。

**修复时闪一下？** → 会，0.15 秒正常现象。
